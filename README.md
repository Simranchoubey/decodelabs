# 🌟 DecodeLabs Project 1

A modern, responsive web application built as part of the DecodeLabs Frontend Development Program. This project demonstrates strong fundamentals of web development, focusing on clean UI design, responsiveness, accessibility, and user experience.

## 🚀 Live Demo

🔗 Live Website: https://simranchoubey.github.io/decodelabs/

## 📖 Overview

This project showcases modern frontend development practices using HTML, CSS, and JavaScript. The application is designed to provide an intuitive user experience while maintaining clean code architecture and responsive layouts across devices.

## ✨ Features

* 🎨 Modern and responsive user interface
* 📱 Mobile-first design approach
* ⚡ Fast loading and optimized performance
* ♿ Accessibility-friendly components
* 🔥 Interactive user experience
* 🌐 Cross-browser compatibility
* 📐 Clean and structured layout
* 🎯 User-centric design principles

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/SimranChoubey/decodelabs.git
```

### Navigate to the Project

```bash
cd decodelabs
```

### Open Locally

Simply open `index.html` in your browser or use a local development server.

For VS Code users:

```bash
Live Server → Open with Live Server
```
